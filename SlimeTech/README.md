# #00 引言

## 概述

本章旨在旨在让读者能够有能力设计绿萌机器，**至至至少**完成一次绿萌扫盲，能够分析简单的绿萌机器。
本书目前仅仅基于Molforte本人一人之拙见，**不代表整个绿萌圈的看法与设计思路**，希望读者能辩证阅读。
